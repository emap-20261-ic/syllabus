#+title: Laboratório 3

* Conteúdo programático

- uso de arrays
- compilação de programas C
- debug de erros
- uso de strings
- uso dos parâmetros de linhas de comando
  
* Objetivos

1. praticar o uso das ferramentas: linha de comando, editor,
   compilador etc
2. Praticar o uso dos tipos de dados básicos em C
3. Familiarização com as estruturas sintáticas básicas da linguagem C

- revisão conceitos

divisão de inteiros e conversão de tipos.
