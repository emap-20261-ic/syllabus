* Programação estruturada (slides-5)

** Algoritmos e estruturas

Um programa de computador executa uma sequência de instruções para
realizar uma tarefa. O conjunto dessas instruções e como elas são
organizadas é o que chamamos de algoritmo.

Na programação estruturada, as instruções são classificadas em três
tipos de estrutura:

- *Sequência*: instruções executadas uma após a outra, representadas
  em caixas interligadas por setas no fluxograma.

- *Decisão*: desvio condicional — algumas ações só são realizadas se
  uma determinada condição for atendida.

- *Iteração* (repetição ou laço): uma instrução ou sequência é
  repetida um número pré-definido de vezes ou enquanto uma condição
  for satisfeita.

** Fluxogramas

As instruções de um programa podem ser representadas graficamente por
meio de fluxogramas. Mostrar os diagramas de cada estrutura
(sequência, decisão, iteração).

** Exemplo: conversão Celsius para Fahrenheit

$$T_f = \left(T_c \times \frac{9}{5}\right) + 32$$

Construir o fluxograma para este algoritmo simples: receber a
temperatura em Celsius, aplicar a fórmula, exibir o resultado em
Fahrenheit.

construindo fluxogramas.
