* Hardware, software e sistema operacional (slides-3)

** CPU e sistema operacional

A CPU coordena tudo que é executado. Sequências de comandos
armazenados na memória são enviados para a CPU. Mas vários programas
podem rodar ao mesmo tempo — o sistema operacional (SO) gerencia isso,
dando a cada programa uma fração de tempo da CPU.

Exemplos de SOs: Linux, Windows, MacOS, iOS, Android.

** Utilitários e aplicativos

SOs modernos incluem utilitários (gerenciador de arquivos, drivers,
interpretador de comandos) e aplicativos (editores de texto, editores
de imagem, navegadores). Também podemos instalar novos programas
compatíveis com o SO.

** CLI e GUI

- CLI (Command Line Interface): interface texto, o usuário digita
  comandos e recebe texto como resposta
- GUI (Graphical User Interface): botões, menus, janelas, ícones

Neste curso vamos implementar programas CLI.

** Gerenciador de arquivos e shell

O gerenciador de arquivos (Explorer, Finder, Nautilus) manipula
arquivos e diretórios no sistema de arquivos.

O interpretador de comandos (shell) permite interagir com o SO por
linha de comando. Gerencia variáveis de ambiente e pode executar
scripts. Exemplos: CMD, bash, zsh.

Diferença entre gerenciamento de arquivos no desktop versus no iOS —
discussão.

* O que é programação (slides-4)

** Programação é resolver problemas

Programas recebem entradas e geram saídas. Exemplo: calcular raízes
quadradas — a definição matemática permite reconhecer uma raiz, mas
não descreve um procedimento para encontrá-la. Essa é a diferença
fundamental entre matemática e computação: a computação requer um
algoritmo.

** Linguagens de programação

Três tipos gerais:

1. Linguagem de máquina — bits, entendida diretamente pelo hardware,
   dependente de máquina
2. Linguagem assembly — mnemônicos (MOV, ADD, RET), traduzida por um
   assembler
3. Linguagens de alto nível — instruções expressivas, traduzidas por
   compiladores. Exemplos: C, C++, Python

** A linguagem C

Desenvolvida por Dennis Ritchie na Bell Laboratories. C e C++ estão
entre as linguagens mais populares. C99 é o último padrão ANSI para
C. C++ é uma extensão de C (Bjarne Stroustrup, anos 80).

Referência: [[https://youtu.be/G1-wse8nsxY?si=lx0ygn0QGbFg_Bz0][C Programming Language, Brian Kernighan and Lex Fridman]]

Além das linguagens compiladas, existem as linguagens interpretadas
como Python.

** Fases do desenvolvimento em C

Mostrar o exemplo de código C e suas transformações:

1. Edição — escrever código fonte em editor de texto (VS Code, Emacs).
   Arquivos com extensão =.c=.
2. Pré-processamento — diretivas =#include=, junção de arquivos.
3. Compilação — tradução do código fonte para código objeto (linguagem
   de máquina).
4. Ligação (linking) — o linker junta o código objeto com as
   bibliotecas, produzindo o executável.
5. Carga (loading) — o loader transfere o executável para a memória
   RAM.
6. Execução — a CPU executa o programa instrução por instrução.

Mostrar o exemplo da transformação: código C $\to$ assembly $\to$
bytes em linguagem de máquina.
